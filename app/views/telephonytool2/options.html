Your child maintenance options

I am required to talk you through both of your options, regardless of your personal circumstances. This is to make sure you have all the information you need to make the best decision for you and your children.

Your first option is to make your own arrangement

  • this is a private arrangement between you and the other parent
  • no one else will be involved
  • it can be whatever you and the other parent agree to

You and the other parent can:

  • {% if data['choice'] === 'receiving' %} come to an agreement about how much you will be paid {% elseif data['choice'] === 'paying' %} come to an agreement about how much you will pay {% endif %}
  • change the amount if you need to
  • {% if data['choice'] === 'receiving' %} decide how often you will be paid {% elseif data['choice'] === 'paying' %} decide how often you will make payments {% endif %}
  • {% if data['choice'] === 'receiving' %} agree for the other parent to pay for specific things, like school uniforms or trips {% elseif data['choice'] === 'paying' %} agree for you to pay for specific things, like school uniform or trips {% endif %}

Making your own arrangement might be a good option if:

  • you feel safe dealing with the other parent
  • you think the other parent will stick to an agreement

Do you have any questions about making your own arrangement?

Your second option is to use the Child Maintenance Service

  • {% if data['choice'] === 'receiving' %} using the Child Maintenance Service is less flexible than making your own arrangement because the amount is based on the paying parent’s earnings {% elseif data['choice'] === 'paying' %} using the Child Maintenance Service is less flexible than making your own arrangement because the amount is based on your earnings {% endif %}
  • it might be a good option if you do not feel safe dealing with the other parent or if you have not been able to stick to your own arrangement in the past

You will need to complete an application online or on the phone

  • there is a £20 fee to make an application – you will not need to pay the fee if you are under 19 or if you have experienced any domestic abuse
  • the fee is non-refundable
  • it does not guarantee an arrangement

Then the Child Maintenance Service will:

  • {% if data['choice'] === 'receiving' %} work out how much child maintenance you will be paid {% elseif data['choice'] === 'paying' %} work out how much child maintenance you will pay {% endif %}
  • {% if data['choice'] === 'receiving' %} arrange for the other parent to pay you {% elseif data['choice'] === 'paying' %} arrange for you to pay the other parent {% endif %}
  • take action if payments are not made
  • help find the other parent if you've lost contact
  • recalculate the payment amount if changes in parents' circumstances are reported
  • help if someone denies they're the parent of a child

There are 2 ways you can pay Child Maintenance payments

Direct Pay

  • {% if data['choice'] === 'receiving' %} the other parent will pay maintenance directly to your bank account {% elseif data['choice'] === 'paying' %} you pay maintenance directly into the other parent’s bank account {% endif %}
  • {% if data['choice'] === 'receiving' %} you do not need to have any contact with the other parent {% elseif data['choice'] === 'paying' %} you do not need to have any contact with the other parent {% endif %}
  • there is a £20 application fee to use the Child Maintenance Service, but there are no additional fees if you use Direct Pay

Collect and Pay

  • {% if data['choice'] === 'receiving' %} the Child Maintenance Service will collect maintenance from the other parent and pass it on to you {% elseif data['choice'] === 'paying' %} the Child Maintenance Service will collect maintenance from you and pass it to the other parent {% endif %}
  • {% if data['choice'] === 'receiving' %} there is a £20 application fee to use the Child Maintenance Service and additional monthly fees for you and the other parent if you use Collect and Pay {% elseif data['choice'] === 'paying' %} there is a £20 application fee to use the Child Maintenance Service and additional monthly fees for you and the other parent if you use Collect and Pay {% endif %}
  • {% if data['choice'] === 'receiving' %} you can only start with Collect and Pay if the other parent agrees to it or if they do not respond to us {% elseif data['choice'] === 'paying' %} you can only start with Collect and Pay if the other parent agrees to it {% endif %}

Do you have any questions about using the Child Maintenance Service?
